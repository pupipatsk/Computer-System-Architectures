1.จงอธิบายความแตกต่างระหว่าง Symmetric และ Asymetric encryption ในด้านของการคำนวณที่ต้องใช้ และลักษณะการใช้งาน
: Symmetric Encryption: ใช้ key เดียวกันทั้งในการเข้ารหัสและถอดรหัส ทำให้มีการคำนวณที่เร็วและใช้พลังงานน้อยกว่า
Asymetric encryption: ใช้ 2 key คือ Public Key สำหรับการเข้ารหัส และ Private Key สำหรับการถอดรหัส การคำนวณจะช้ากว่าและจำนวนมากกว่า เนิ่องจากซับซ้อนกว่า

ลักษณะการใช้งาน
Symmetric Encryption: เหมาะสำหรับการเข้ารหัสข้อมูลที่ต้องการความเร็วและประสิทธิภาพในด้านการถ่ายโอนข้อมูล e.g., VPN, Internal in Application
Asymmetric Encryption: เหมาะสำหรับส่งผ่าน public e.g., Authentication, HTTPS, TLS


2.จงยกตัวอย่างปัญหา security ที่เป็นไปได้ จากการใช้ hardware ที่ถูกผู้ไม่ประสงค์ดีเปลี่ยนแปลง
: Hardware Trojans

ฝังวงจรขนาดเล็กที่ทำหน้าที่เหมือนกับประตูหลัง (backdoor) ที่ผู้โจมตีสามารถเรียกใช้งานได้ 
โดยฮาร์ดแวร์นี้อาจทำงานเมื่อรับคำสั่งหรือข้อมูลเฉพาะบางชุดเท่านั้น 
ซึ่งทำให้การตรวจจับทำได้ยาก เช่น 
อาจมีวงจรที่เปลี่ยนสิทธิการเข้าถึงของโปรเซสเซอร์ให้กลายเป็นโหมดผู้ดูแลระบบเมื่อได้รับโค้ดที่กำหนด 
ทำให้ผู้โจมตีสามารถเข้าควบคุมระบบได้เต็มที่​


3.Object ถูกเก็บไว้ในบริเวณใดของmemoryของโปรแกรม
: Heap


4.จงเล่าถึงการโจมตีที่นิสิตประทับใจที่สุดในคาบเรียนที่ผ่านมา พร้อมเหตุผล
: Rowhammer Attack

เพราะสามารถโจมตีDRAMโดยใช้การรั่วไหลของกระแสไฟฟ้า ซึ่งไม่ต้องรอ permission อะไร เป็นการใช้หลักการของ physicalโดยตรง ที่ไม่ได้รออาศัยช่องโหว่ของ software
